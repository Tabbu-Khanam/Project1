What is HTML?

Html stands for hypertext markup language.It's standard markup language for creating web pages.HTML consists of a series of elements.HTML elements tell the browser how to display the content.With HTML you can create your own websites.

What is CSS?

CSS stands for cascading stylesheets.CSS describes how elements are to be display on screen,paper,in other media.CSS saves a lot of work.It can control the layout of multiple web pages all at once.
